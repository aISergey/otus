#### Создадим новую БД
- войдём под postgres:
> su postgres
- запустим psql
> psql
- создадим новую БД:
> create database TEST;
- сменим текущую БД:
> \c TEST
#### Создадим таблицу и заполним её:
```
create table test(c1 citext);
insert into test values('тест 1');
\q
```
